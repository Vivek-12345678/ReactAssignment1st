1.Virtual DOM

In React, for every DOM object there is a corresponding "virtual DOM object".A virtual DOM object is a representation of DOM object,like a lightweight copy.

A virtual DOM object has the same properties as real DOM object but it lacks the real thing's power to directly change what's on the screen.

Manipulating the DOM is slow while virtual DOM is fast beacuse nothing gets drawn on screen.


2.SPA

Single Page Application is a web app implementation that loads only single web document, and then updates the body content of that single document via a JavaScript APIs such as XMLHttpRequest and fetch when different content is to be shown


3.Difference between class and functional component

Functional components are written as a JavaScript function whereas Class components as a JavaScript class

Functional components do not have a state or lifecycle methods whereas Class component have a state and can implement lifecycle methods like Mounting,Updating,Unmounting

Fnctional Component are faster as they do not have state and lifecycle whereas Class Components are slower as they have state and lifecycle

5.JSX and Why do we use it instead of js

JSX stands for JavaScript XML

JSX allow us to write HTML elements in Java Script and place them in DOM without any createElement() or appendChild methods

JSX converts HTML tags into react element so it makes it easier to write React Application


6.What is Package.json?

Package.json is a metadata file that lists all the dependencies required for the project to run.It includes information such as the name and version of the project, dependencies and their respective version
